## 📋 About

Interactive CV/resume with a clean neon-cyberpunk aesthetic, dark/light theme toggle, and fully responsive layout. Built with HTML, CSS, and JavaScript.

## ✨ Features

- Dark/Light theme toggle with local storage persistence
- Animated ambient glow effects
- Print-to-PDF support for downloading as a professional CV
- Fully responsive design
- Skills, projects, education, coursework, and honors sections
